## ğŸ†• How to Create a Collection First

### If You Don't Have a Collection Yet:

### Step 1: Create Collection

1. **Look at left sidebar**
2. **Click "+" button** or **"New"** button
3. **Select "Collection"**
4. **Name it** (e.g., "Supabase API")
5. **Click "Create"**

### Step 2: Add Requests to Collection

1. **Right-click on a request** (or click three dots)
2. **Click "Move"** or **"Add to Collection"**
3. **Select your collection**
4. **Repeat for all requests**

### Step 3: Now Export

- Use any method above to export
