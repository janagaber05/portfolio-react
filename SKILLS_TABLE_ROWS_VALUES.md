## üìù Important Notes

1. **Icon Type**: 
   - Only **Figma** (Row 5) needs `icon_type = "figma"`
   - Only **Adobe** (Row 7) needs `icon_type = "adobe"`
   - All others leave `icon_type` empty

2. **Name**: 
   - Use exact abbreviations: `PS`, `PR`, `AN`, `LR`, `AE`, `AI`, `CSS`, `HTML`, `JS`
   - Use full name for: `Figma`, `Adobe`

3. **Display Order**: 
   - Must be unique numbers from 0 to 11
   - This determines the order they appear in your skills matrix

4. **Optional Fields**: 
   - `name_ar`, `short_name`, `category`, `proficiency` can all be left empty
   - Only fill them if you need Arabic names or additional info

5. **Auto-generated**: 
   - Don't fill `id`, `created_at`, or `updated_at`
